Superfeedr is PubSubHubbub with benefits

It looks like you're a PubSubHubbub subscriber. You implemented this protocol in your app and you're now able to subscribe to feeds in realtime. Superfeedr hosts many of these hubs, like the one for Tumblr, Posterous, or Gowalla.

PubSubHubbub is great, but we know that it doesn't solve all the problems. How do deal with the majority of feeds that are not PubSubHubbub yet? If you used Superfeedr's default hub, you can get notification from any feed.

No more polling

Polling is an option. Polling is also hard to scale and there exists several technologies out there to determine when a given feed has been updated. Superfeedr implements a lot of them so you don't have too. RSSCloud, SUP, XML-RPC ping, custom streaming APIs? We've got you covered. We keep the complexity on our end and push content to you exactly as if it was all PubSubHubbub. You can also get content pushed to you with XMPP.

Data Normalization

You actually already benefit from this, but Superfeedr does content normalization. You probably know it already, but it's a pain to get content in so many different formats, and often times invalid. Superfeedr deals with this complexity on your behalf. No matter the original format, we map this into some clean Atom, with a hint of georss for geo-location and ActivityStreams for any social web data.

Digest and Querying

As any other Publish-Subscribe pattern, PubSubHubbub fails silently. Ever wondered what happens if the hub to which you subscribed died? On the other hand, what happens if, for some reason your app has been offline for a few hours? Don't worry, Superfeedr offers digest notifications which will guarantee you that you'll get content. You can also query Superfeedr to get the last 10 entries of a given feed. This way, you won't miss any data.

Track

Sometimes, subscribing to a feed is not enough. You want to have any content that matches a keyword. Superfeedr does that. It's called track. You can get a notification with any entry that matches this keyword. Want to monitor the number of mentions of your companies name, technologies that you use, compare popularity? Track is for you.

Get started

Signup for Superfeedr. The only you have to do is point to http(s)://user:password@push.superfeedr.com for the hub url. Yes, that's it.
